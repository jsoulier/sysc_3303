=== Setup Instructions ===

1. Start IntelliJ IDEA
2. Open the Assignment1_101189819 folder in IntelliJ IDEA
3. In the project explorer, right click on src/Main.java and click "Run"

=== Files ===
 
src/Barista.java
 - A class that represents a barista thread that takes ingredients from the supplier.

src/Ingredient.java
 - An enumeration that represents the available ingredients.

src/Main.java
 - A class that creates the baristas, supplier, and provides the entrypoint of the application.

src/Supplier.java
 - A class that represents a supplier thread that provides the baristas with ingredients.
